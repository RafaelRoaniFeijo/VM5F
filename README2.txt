Elaboração de leito r/gravador de tags veiculares. A partir de um Orange Pi Pc e um módulo uhf VM5 F. A realização do firmware foi adaptada para funcionamento no Orange Pi. Foi necessário conhecimento dos comandos executados diretamente na memória interna do VM5F. Esse leito r/gravador identifica tags a uma certa distância, os lê, registra em um arquivo de texto somente de leitura e grava em um arquivo somente de escrita temporariamente. Está sendo ainda desenvolvido o front e backend, será realizado a partir do software QTcreator, para controle em um display touch screen. Todo projeto sendo desenvolvido no shell script bash e zsh e 
utilizando Makefiles e elaboração de um script em python para iniciar o programa sempre que ligasse o sistema (funcionar como um firmware).
